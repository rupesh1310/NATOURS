## 🚀&nbsp; Installation

`NodeJs Application`

`MongoDB`

`MongoDB Atlas`

`npm`

`POSTMAN`

## :pencil: &nbsp; **Import and Delete data from dev-data folder**

**Delete Data**

`node .\dev-data\data\import-dev-data.js --delete`

**Import Data**

`node .\dev-data\data\import-dev-data.js --import`

## ✅&nbsp; Requirements
 `Latest version of node`

`MongoDB Atlas Account`

`Mailtrap Account`

## :wrench: How to use
**Import Data**
> node .\dev-data\data\import-dev-data.js --import

**Start in development**
> npm run start:dev
> npm run watch:js

**Start in debug**
> npm run debug

**Start in production**
> npm run start:prod

**Database**
> MongoDb Atlas
