# How to use the application : 

**import and delete data from dev-data folder**

**Delete data**
node .\dev-data\data\import-dev-data.js --delete

**import data**
node .\dev-data\data\import-dev-data.js --import

**start in development**
npm run start:dev
npm run watch:js

**start in debug**
npm run debug

**start in production**
npm run start:prod

**Database**
MongoDb Atlas
