## 🚀&nbsp; Installation

`> NodeJs Application`
`> MongoDB`
`> MongoDB Atlas`
`> npm`
`> POSTMAN`

**import and delete data from dev-data folder**

**Delete data**
node .\dev-data\data\import-dev-data.js --delete

## ✅&nbsp; Requirements
> Latest version of node
> MongoDB Atlas Account
> Nodemailer Account

## :wrench: How to use
**Import Data**
> node .\dev-data\data\import-dev-data.js --import

**Start in development**
> npm run start:dev
> npm run watch:js

**Start in debug**
> npm run debug

**Start in production**
> npm run start:prod

**Database**
> MongoDb Atlas
